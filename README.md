# Intallation

## Requirements

- Python 3.10+
- just
- uv

## Installation

```bash
just install
```

## Configuration

### Remote host

Set the remote host in `sample.env` and rename it to `.env`.

### Remote user

Set the remote user in `sample.env` and rename it to `.env`.

### Remote workspace

Set the remote workspace in `sample.env` and rename it to `.env`.
